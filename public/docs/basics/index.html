The Docs: Basic Usage

Let's get started

Seeing a live view

Navigate to /live to see a live view of what the server's page is currently at and enjoy some fun tunes

Admin View

If you are a website admin, you will have access to the /admin pages.

You will have the ability to fully customize the server through these pages.

You can use the page creator in the lower left hand corner on the admin page to create your very own nice looking pages to display.

Make sure to save after you make changes with the save button in the lower right hand corner.  Note that after adding a page it will automagically save all changes.
