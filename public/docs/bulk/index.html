The Docs: API

Bulk

The bulk API is used when you're trying to know everything about the server.  It lists every category the server has, every page, every playlist, when the next category will be displayed, and when the next page will be displayed.  It's your one-stop-shop.

The bulk api can be accessed admin-restriction free at:

/api/bulk

Admin Bulk Request

This request will require a cookie containing a token of authentication in order to function. Without this token, you will be redirected to the login page. This operation is a POST request, and will modify data on the server.  This request is used when modifying data about categories, playlists, show times, page names, and everything else that isn't uploading a page.
