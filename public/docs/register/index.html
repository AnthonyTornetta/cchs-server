The Docs: Register

Let's get started

How to register

In order to register an account, you must be granted permission by another account holder by having them also enter their credentials

Registering without code

To login just using this site, navigate to /register

Registering with code

In order to register an account, a POST request must be made to /register, with the body containing:

  • The new account's username, called "username" in the body
  • The new account's password, called "password" in the body
  • The authorizing account's username, called "authUsername" in the body
  • The authorizing account's password, called "authPassword" in the body

If the registration is successful, a token will be given and the account will have been created. This token functions the same as a login token would.

If the attempt is unsuccessful, an error message will be given.
