The Docs: Render

2D Rendering Engine

Rendering MarkDown Files

By making a GET request and specifying the category and page to render, you will get the rendered out HTML version of the markdown file.

Usage:

/api/render?page=page-name&category=category-name

Rendering MarkDown Text

Make a POST request to the same location, and put the text you want to render in the body under the "src" tag.
