The Docs: Song

Sound Engine

Getting Songs

By making a GET or POST request and specifying the youtube ID of a song, the server will do its best to get you that song.

If the server does not have that song on file, it will make it appear out of thin air, though this may take about 10-40 seconds depending on the song's length, leading to some long request times.

Usage:

/api/song?id=dQw4w9WgXcQ
