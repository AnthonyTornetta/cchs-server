The Docs: Status

Status

The status API is used when you're trying to get a basic overview of what page the server is on, what playlist its playing, and what category the server is displaying (and its settings).  The status request will also show you the server's next category time and next page time.

The status can be accessed without admin restriction through

/api/status

The data returned will be in JSON format.
